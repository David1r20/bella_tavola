# Bella Tavola — Prova P1

Projeto completo para a P1 de CDIA, integrando FastAPI, Machine Learning e CI/CD.

## Estrutura do Projeto

- `/routers`: Lógica das rotas da API.
- `/models`: Definições de esquemas Pydantic.
- `/ml`: Geração de dados e treinamento do modelo de fraude.
- `/tests`: Testes automatizados com Pytest.
- `.github/workflows`: Automação de CI.


## Instruções de Uso
1. Instale os requisitos: `pip install -r requirements.txt`
2. Execute a aplicação: `python main.py`
3. Acesse `http://localhost:8000/docs` para interagir com a API.
